The primary API interfaces. These services provide the necessary methods to access and manipulate OpenMRS domain objects.

Services are obtained from a org.openmrs.context.Context (@see org.openmrs.context.Context).

For example: 

  Context context = ContextFactory.getContext();
  PatientService ps = context.getPatientService();

  // fetch patient with identifier 3-4
  Patient patient = ps.getPatientByIdentifier("3-4");
